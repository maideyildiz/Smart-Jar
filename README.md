# Smart-Jar
This is a easy IoT project that can track your jar's occupancy ratio.

For this project I used Hc-SR04 Ultrasonic Sensor (which is not really the best choice but it's cheaper),Arduino Uno, Nodemcu ESP8266 and jumper cables.
(Instead of Nodemcu you can also use Esp8266 wifi module that one is cheaper but I already had Nodemcu so I used what I have.)

All connection is on the Connection.pdf file that I upload here.

Ones you connect everything first load code to Arduino, then load the code to Nodemcu without connecting the RX TX pins.

When code loaded connect RX,TX pins and you good to go!
